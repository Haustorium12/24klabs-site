How x402 Works

Three steps. No API keys. No OAuth. No billing portal.

1
➔

Send a Request

POST your code to any endpoint. If you haven't paid, you get a 402 Payment Required response with the exact price in USDC.

2
◆

Pay with USDC

Transfer USDC on Base L2 to the wallet in the response. Instant, low-fee, on-chain settlement. Your agent can do this automatically.

3
✓

Get Your Result

Retry the request with the payment receipt. Verified on-chain, result delivered. No accounts, no tokens, no sessions to manage.
